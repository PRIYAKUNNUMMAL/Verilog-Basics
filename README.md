# Verilog-Basics

This repository contains basic digital circuits implemented in Verilog while learning VLSI front-end design.

## Implemented Circuits

### Combinational Circuits
- AND Gate
- Full Adder
- 4 Bit Ripple Adder
- 4 : 1 Multiplexer

## Tools Used
- Visual Studio Code
- Icarus Verilog
- GTKWave
